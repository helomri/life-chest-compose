# Life Chest

A complete rewrite of my application Life Chest in Jetpack Compose with more features
and better performance.

## Feature list

- [x] Multiple vaults
- [x] Onboarding
- File playback
  - [x] Video
  - [x] Audio
  - [x] GIF
  - [x] Images
- [x] File sorting
- [x] Directories and subdirectories
- [x] File export/sharing
